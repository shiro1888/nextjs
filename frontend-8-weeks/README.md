# Frontend 8 Weeks

开始日期：2026-07-03

目标：8-10 周后具备前端实习投递能力，并能接简单静态建站小单。

## 目录

```text
week-01-html-css-js/      HTML/CSS + JavaScript 起步
week-02-js-core/          JavaScript 数组、对象、函数
week-03-dom-async/        DOM、事件、异步、fetch
week-04-react-todo/       React Todo
week-05-react-movie-search/ React API 项目
week-06-react-ts-cart/    React + TypeScript 购物车
week-07-nextjs-starter/   Next.js App Router 入门
week-08-nextjs-final-project/ Next.js 毕业项目
notes/                    每日学习日志
```

## 每日纪律

1. 每天写代码
2. 每天写日志
3. 每天至少一次 commit
4. 卡住 30 分钟就整理问题求助
